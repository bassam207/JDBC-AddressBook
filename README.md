# JDBC Address Book

## Project Description
This is a Java backend project that demonstrates how to work with JDBC and PostgreSQL.  
It uses DAO (Data Access Object) pattern to interact with the database and allows you to perform basic CRUD operations.

## Features
- JDBC connection to PostgreSQL
- DAO pattern implementation
- Batch insertion of multiple contacts
- PreparedStatement usage for safer SQL queries

## Project Structure
- `src/main/java/org/ContactPerson.java` → POJO class representing a contact
- `src/main/java/org/ContactPersonDao.java` → DAO class for database operations
- `src/main/java/org/Main.java` → Main class to run the application
- `src/main/resources/db.properties` → Database configuration file
